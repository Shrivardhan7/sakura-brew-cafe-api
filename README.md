# Sakura Brew Café API

A Japanese-themed café ordering REST API built with Node.js, Express, and MongoDB.

## Features

- JWT Authentication
- Menu system
- Vegetarian drinks and snacks
- Order management
- Multilingual menu (English / Japanese)
- Docker support
- Jest API testing

## Tech Stack

Node.js  
Express  
MongoDB  
JWT  
Docker  
Jest  

## Installation

git clone https://github.com/Shrivardhan7/sakura-brew-cafe-api.git  
cd sakura-brew-cafe-api  
npm install  

Create `.env`

PORT=5000  
MONGODB_URI=mongodb://localhost:27017/sakura-brew  
JWT_SECRET=secret  

Run server

npm run dev

## API

http://localhost:5000/api/menu
